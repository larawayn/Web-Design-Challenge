Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes based on latitude and distance from the equator. Using OpenWeatherMap API, weather and location data was first pulled and used to assemble a dataset of over 500 cities. The data was brought into Python and put into dataframes using Pandas. Matplotlib was then used to plot the weather attributes vs. latitude. Weather attributes included: temperature, cloudiness, wind speed and humidity. This site contains the source data, visualizations and summary of the trends and correlations observed.
